# 🧠 Builder Agent Spec – Framer Motion Audio Chat Demo Page

## 🎯 מטרת הדף
הדף מיועד לשמש **עמוד פרסומי / תדמיתי** המציג את פרויקט  
**Framer Motion – Audio Chat Demo**  
כדוגמה ליכולות של Framer Motion בעיצוב חוויית משתמש דינמית.

המטרה אינה להציג מערכת עובדת, אלא **להדגים ולפרסם** את הפוטנציאל של שילוב בין  
אנימציות מתקדמות, ממשק צ׳אט, ועריכת אודיו אינטראקטיבית.

---

## 🧩 מטרות הפיתוח

1. להמחיש את השימוש ב־**Framer Motion** בממשק עשיר ואינטראקטיבי.  
2. להציג בצורה מושכת את **חזון המערכת** — שילוב בין שיחה, תנועה ועריכת קול.  
3. לשמש כ־**עמוד פרסומי לפרויקטים עתידיים** או דוגמאות בתיק עבודות.  
4. להיות **עמוד עצמאי רספונסיבי**, שניתן לשלב ב־Vercel או בכל אתר קיים.

## 🧭 יעדי הביצוע לסוכן

1. לבנות את דף הפרויקט ב־Next.js תוך שימוש ב־Framer Motion לאנימציות כניסה ו־hover.  
2. להגדיר מבנה רספונסיבי מלא בעברית (RTL).  
3. להכניס את כל הטקסטים המופיעים במסמך זה למקומם בדף.  
4. לייצר גרסה מוכנה לפריסה ב־Vercel.  
5. לדאוג למבנה קוד קריא עם רכיבים נפרדים לכל מקטע.  
6. לשמור על קו עיצוב כהה ומינימליסטי (ללא צבעים עזים).  
7. לבדוק שכל המעברים פועלים חלק (motion.div / AnimatePresence).  
8. להוסיף תגיות מטא בסיסיות לקידום במנועי חיפוש.
