## WaveQ - Landing Page

## 🎯 סקירה כללית

**מטרת הפרויקט:** בניית עמוד פרסומי/תדמיתי ב-Next.js המציג את פלטפורמת WaveQ - מערכת אינטראקטיבית של צ'אט ועריכת אודיו עם אנימציות מתקדמות.

**סוג הפרויקט:** Landing Page / Demo Page  
**שפת פיתוח:** TypeScript + React + Next.js  
**סגנון עיצוב:** Dark Mode, מינימליסטי, RTL מלא  
**פלטפורמת פריסה:** Vercel
